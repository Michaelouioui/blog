## 项目初始化

先在 gitee 创建一个新的仓库

勾选Javascript、VS code、第3个许可证、开源、readme

在电脑上创建文件夹，将项目克隆过来

进入VS code 将项目拖进来

gitignore 添加不通过git进行备份的文件夹

```
#npm包
/node_modules/

#输出
/dist/
```

进行npm init

## Webpack

在项目根目录创建src目录，在src里面创建文件夹 view 、page、images

进行webpack的安装

```bash
npm install --save-dev webpack
npm install --save-dev webpack-cli
```
